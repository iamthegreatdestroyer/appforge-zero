## ğŸš€ Getting Started

### Prerequisites

- **Node.js** 20.x LTS
- **Python** 3.11+ (for trend analysis)
- **Android SDK** API Level 34
- **Java** JDK 17+ (for Gradle)

### Installation

```bash
# Clone the repository
git clone https://github.com/your-org/appforge-zero.git
cd appforge-zero

# Install dependencies
npm install

# Setup Python environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install pytrends praw

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Start development server
npm run dev
```

### First App in 60 Seconds

```bash
# 1. Launch AppForge Zero
npm run dev

# 2. Select a template (e.g., Wallpaper Pack)
# 3. Enter theme: "Cyberpunk Neon City"
# 4. Click "Generate"
```
